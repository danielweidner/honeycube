The Game Engine you have been waiting for — not!

HoneyCube is not an attempt to reinvent the wheel. There are a variaty of good game engines for different purposes available in the market. Unity3D, OGRE, Irrlicht, just to name a few. Why bother with the HoneyCube Engine then?

HoneyCube as a learning plattform

The project is an attempt to learn more about the fundamentals of game programming. We want to provide a look behind the curtain and play arround with the different technologies involved. The project site you are looking at thereby will document the overall learning process together with the provided source code.

You are welcome to contribute!
